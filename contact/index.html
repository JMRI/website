Contact Information

Email discussion group

There is an active "Yahoo Groups" user community called JMRI users.

This discussion group is for people using the JMRI model railroad software.
It is a good place to discuss any aspect of these. If you have questions about how to do things, suggestions, or even complaints, post them here.

In addition to the above user community, there are several developer focused mailing lists hosted on sourceforge.net that you may find useful:

  • jmri-announce Archives (Subscribe)  Announcements of new JMRI versions.
  • jmri-commits Archives (Subscribe)  CVS/Git commit notifications.
  • jmri-developers Archives (Subscribe)  Discussion of current and future development.

All of the JMRI code is maintained on GitHub. See the:

  • JMRI/JMRI GitHub Issue List (Subscribe)  Repository for notes on work in progress.

  • Contact us via the JMRI users mailing list

Thanks and congratulations to all who contributed!
