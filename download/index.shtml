Test version 4.5.5 released

Test version 4.5.5 was made available on October 17, 2016.

JMRI 4.5.5 is the next test release in the current development series, containing the latest and greatest JMRI updates.

The series is working toward the next JMRI production version, probably in late 2016.

For more information, including the links to download a copy, please see the release note. Version 4.4 is the current "production" version, recommended for first-time users.

Production version 4.4 released

On June 30, 2016 production version 4.4 was made available.

Version 4.4 is the current "production" version, recommended for first-time users. Please see the next section for download and install information.

Production version 3.10.1 released

On January 11, 2015 production version 3.10.1 was made available.

Version 3.10.1 is recommended for JMRI users with computers that can only run Java 1.6. Please see the release note for download and install information.

Production version 2.14.1 released

JMRI - Developer releases

As the program develops, we also produce test versions. You may want to try these, as new features will show up in them first. They are announced in the JMRI users Yahoo group.

The most current test release is JMRI 4.5.5. More information on it, including the links to download a copy, can be found in release note.

For more information on the rough development schedule, please see the JMRI road map (which is only occasionally updated).

CATS

The Crandic Automated Traffic System (CATS) is a JMRI-based application by Rodney Black for controlling and operating a model railroad. There's more information on the CATS web site.

CATS 2.34 (Release2037) is compatible with JMRI 3.3.1 thru 3.8. We recommend that you use JMRI 3.8 with it.
